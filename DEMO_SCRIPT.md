# Demo Script

1. Open Home.
   Expected result: demo data, mock adapters and module status are visible.
2. Open Data Import Center.
   Expected result: BOM, WMS inventory, customer orders and shipment exports are classified.
3. Open BOM & Inventory Dashboard.
   Expected result: material coverage and risk status are calculated for `FG-OPS-A100`.
4. Open Product Material Trace.
   Expected result: finished product connects to components, source rows, inventory, inbound records and suppliers.
5. Open Release Notice Generator.
   Expected result: a release notice preview is generated from product, order and material coverage.
6. Open Production Line Simulation.
   Expected result: `LINE-UNIVERSAL-A` shows machines, links, states and 24h simulation values.
7. Open Simulation Report.
   Expected result: bottleneck machine, quality bottleneck, output, scrap, waiting time and blocking time are displayed.
8. Open Operations Q&A.
   Expected result: the response includes tool trace and source-backed evidence.
9. Open Agent Trace.
   Expected result: question, workflow, tools and timestamp are recorded.
10. Open Integration Status.
    Expected result: ERP, WMS, MES, PLC, scheduling and MCP modes are listed.
