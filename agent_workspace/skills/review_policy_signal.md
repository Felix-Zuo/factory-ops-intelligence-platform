# Skill: Review Policy Signal

Use when the user asks about customs, tariff, import/export, regulatory or policy changes.

## Tools

1. `search_policy_signals`
2. `product_material_trace`
3. `generate_decision_brief`

## Guardrails

- Prefer official-source links and source metadata.
- Keep source facts separate from model summaries.
- Link signals to materials, products and orders before recommending release actions.
- Keep compliance review human-owned.
