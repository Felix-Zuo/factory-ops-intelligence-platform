# Demo Data

All records in this folder are synthetic and safe for public display.

The default scenario follows `FG-OPS-A100`, a generic modular control kit. The wider data pack also includes a sensor pack, fluid service kit and inspection fixture so the demo is not tied to one product category.

The data connects BOM components, WMS-like stock, customer orders, inbound shipments, supplier delivery context, file import logs and configurable line definitions.
