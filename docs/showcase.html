Problem

Operating decisions often sit across ERP records, WMS stock exports, spreadsheet BOM files, shipment notes, supplier updates and message threads. The practical issue is the time spent reconciling source rows before a release decision.

Release notices depend on template discipline, material gates, order due dates and line assumptions. Line output and bottlenecks are easier to review when takt assumptions and machine states share the same view.

MODULE 01

BOM & Inventory Intelligence

MODULE 02

Release Notice Generator

The notice preview is built from product selection, order quantity, BOM rows, material gate status, source references and template version. The public demo exports HTML/JSON previews and keeps watch items visible.

MODULE 03

Production Takt & Bottleneck Simulation

The simulation represents machine nodes, transport links, buffers, running and blocked states, 24h output, utilization, waiting time, blocking time, scrap output and bottleneck detection. Scheduling remains an adapter boundary rather than a shipped optimizer.

Agent Q&A With Tool Evidence

Questions are routed through registered operations tools. The Q&A panel shows the user question, selected intent, workflow, tool calls, result, source/demo record references and final answer.
