QUALITY GATES

Maintained like a product, not a throwaway demo.

The repo is designed for public review: synthetic data, bounded API inputs, explicit adapter modes, CI validation, dependency audit and documented product standards.

Validation

Seed, snapshot, self-check, pytest, smoke demo, tone scan and frontend build.

Security

No credentials or private factory files. API payloads are bounded and CORS is local-only.

Model Boundary

Forecasting is deterministic by default. TimesFM and LLMs are optional adapters, not hidden truth sources.

Roadmap

GitHub milestone tracks scenario profiles, forecast backtesting, policy adapters, visual regression and security posture.
